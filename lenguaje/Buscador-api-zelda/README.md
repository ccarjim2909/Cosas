# Buscador API Zelda

Proyecto web estatico inspirado en The Legend of Zelda.

## Que incluye

- `index.html`: buscador principal conectado con la API de Zelda.
- `favoritos.html`: pagina para ver y gestionar favoritos.
- `styles.css`: estilo compartido entre las dos paginas.
- `common.js`: configuracion de la API y utilidades de favoritos.
- `app.js`: logica del buscador.
- `favoritos.js`: logica de la galeria de favoritos.

## Como usarlo

1. Abre `index.html` en el navegador.
2. Escribe un nombre o cambia la categoria para consultar la API.
3. Pulsa la estrella para guardar resultados en favoritos.
4. Entra en `favoritos.html` para verlos todos con el mismo estilo.

## Nota

Los favoritos se guardan en `localStorage`, asi que permanecen en el navegador del mismo equipo hasta que los elimines.
